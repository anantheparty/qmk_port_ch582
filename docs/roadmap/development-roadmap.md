# Obey65 三模键盘开发路线图

## 概述

本文档规划 Obey65 键盘从当前的有线模式到完整三模（USB + BLE + 2.4G）的开发路径。

## 当前基线状态

- ✅ USB 有线模式：完全工作
- ✅ RGB Matrix (双灯带)：完全工作
- ✅ VIA 支持：已配置
- ❌ 蓝牙模式：有部分代码但未完成，当前编译失败
- ❌ 2.4G 模式：仅桩代码
- ❌ 2.4G 接收器：未开始
